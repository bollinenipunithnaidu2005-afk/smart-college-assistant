# 🎓 Smart College Assistant

A real-world Python-based application designed to manage student data, tasks, and performance analysis.

## 🚀 Features
- Student Management System
- Duplicate Roll Number Prevention
- Topper Detection
- Performance Analysis (Average + Weak Students)
- File-based Data Storage

## 🧠 Key Highlights
- Designed with real-world logic
- Data validation implemented
- Structured modular code
- Insight generation (not just CRUD)

## 🛠️ Tech Stack
- Python
- File Handling

## ▶️ How to Run
python main.py
